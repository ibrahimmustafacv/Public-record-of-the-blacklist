## 📖 نبذة عن المشروع

**السجل العام للقائمة السوداء** هو منصة ويب بواجهة عربية كاملة (RTL) بهوية بصرية فاخرة تجمع بين اللون الكحلي العميق والذهبي، مخصصة لعرض سجل موثّق للأماكن والكيانات (مطاعم، كافيهات، متاجر إلكترونية، صفحات سوشيال ميديا... إلخ) التي تم إدراجها ضمن قائمة سوداء، مع نظام بحث وفلترة فوري، وآلية إبلاغ سهلة وسريعة عن أي كيان جديد لمراجعته.

> 🎯 **الهدف:** توفير مرجع عام شفاف يساعد المستخدمين على تجنّب التعامل مع كيانات موثّقة بسلوكيات مخالفة، مع إتاحة قناة سهلة وسريعة للإبلاغ عن أي كيان جديد.

---

## ✨ أبرز المزايا

| الميزة | الوصف |
|---|---|
| 🔍 **بحث فوري** | بحث لحظي بالاسم أو العنوان دون إعادة تحميل الصفحة |
| 🗂️ **فلترة حسب التصنيف** | تصفية النتائج حسب نوع الكيان (مطعم، كافيه، فندق، متجر إلكتروني...) |
| 🖼️ **عرض تفصيلي لكل كيان** | نافذة تفاعلية تعرض كافة بيانات الكيان مع إمكانية تكبير الصور |
| 📝 **نموذج إبلاغ متكامل** | إرسال بلاغات جديدة مع صورة توضيحية ورابط الموقع الجغرافي |
| 🗄️ **سجل مُحدَّث لحظياً** | عرض حي ودقيق لكل الكيانات المُدرجة في القائمة |
| 🎨 **تصميم فاخر ومتجاوب** | هوية بصرية رسمية (كحلي + ذهبي) متجاوبة بالكامل مع جميع الأجهزة |
| 🌙 **دعم كامل للعربية RTL** | بُنية وتصميم مخصصان للغة العربية من الاتجاه إلى الخطوط |
| ♿ **إمكانية وصول** | دعم لذوي الاحتياجات الخاصة وتقليل الحركة عند الطلب |

## 🖥️ لمحة عن الاستخدام

1. تصفّح السجل العام للكيانات المحظورة مباشرة من الصفحة الرئيسية.
2. استخدم شريط البحث والفلترة للوصول إلى كيان معيّن بسرعة.
3. اضغط على أي بطاقة لعرض كامل تفاصيلها.
4. لديك كيان تريد الإبلاغ عنه؟ استخدم زر **"الإبلاغ عن كيان جديد"** وقدّم البلاغ ليخضع للمراجعة.
